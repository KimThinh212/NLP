# 📘 Thực hành môn Xử lý Ngôn ngữ Tự nhiên (NLP)

## 📌 Giới thiệu

Tài liệu này tổng hợp các bài thực hành trong môn **Xử lý Ngôn ngữ Tự nhiên (Natural Language Processing - NLP)**. Mục tiêu của các bài thực hành là giúp sinh viên hiểu rõ cách xử lý, phân tích và xây dựng các mô hình học máy liên quan đến dữ liệu văn bản.

---

## 🎯 Mục tiêu môn học

* Hiểu bản chất dữ liệu văn bản và các phương pháp biểu diễn
* Nắm vững các kỹ thuật tiền xử lý văn bản (text preprocessing)
* Áp dụng các mô hình học máy và Deep Learning trong NLP
* Xây dựng pipeline xử lý dữ liệu hoàn chỉnh
* Đánh giá và cải thiện hiệu suất mô hình

---

## ⚙️ Công nghệ sử dụng

* Python
* Google Colab / Jupyter Notebook
* Thư viện:

  * numpy, pandas
  * matplotlib, seaborn
  * scikit-learn
  * nltk / spacy
  * tensorflow / pytorch

## 🚀 Hướng dẫn sử dụng

1. Mở Google Colab hoặc Jupyter Notebook
2. Load dataset tương ứng
3. Chạy từng bước trong notebook
4. Quan sát kết quả và trực quan hóa
5. Điều chỉnh tham số để cải thiện mô hình

---

## 📊 Đánh giá

* Accuracy
* Precision / Recall
* F1-score
* Confusion Matrix

---

## 💡 Ghi chú

* Không sử dụng thư viện có sẵn khi được yêu cầu (ví dụ: tự cài Logistic Regression)
* Ưu tiên hiểu bản chất hơn là chỉ chạy code
* Thực hành với dữ liệu thực tế để nâng cao kỹ năng

---

## 📎 Kết luận

Các bài thực hành giúp xây dựng nền tảng vững chắc về NLP, từ xử lý dữ liệu đến xây dựng mô hình. Đây là bước quan trọng để phát triển các hệ thống thông minh như chatbot, phân tích cảm xúc và tìm kiếm thông tin.
